My Projects

My school assignment projects, and also my personal side projects.

AI generated art Classifier

This project is part of my final year project for my university courses. Which primary focus on AI-generated artwork detection researches. The model is trained using dataset of thousands Anime artwork from Pixiv. It is powered by Keras Convolutional Neural Network(CNN) methodology. A web application is served to discover the classification of artworks.

Wood Defect Detector

One of my group assignment of school projects for digital image processing course. It uses openCV for image processing and algorithms to detect the defects like holes and knots of wood in an image. The objective of this assignment is practical in implementations of feature extraction algorithms in digital image processing.

Movie Booking App

My school group assignment project for wireless application development course, An Android app developed by React-native framework. This application emulates the CRUD(create,read,update,delete) process of an online movie booking app
